# Snapshots

Use this folder for stage backups before Git is available or before major restructuring.

Naming convention:

```text
YYYY-MM-DD_stage-name/
```

Examples:

```text
2026-07-01_initial-scaffold/
2026-07-08_pre-literature-import/
2026-07-15_pre-experiment-run/
```

Do not use snapshots as a substitute for evidence tracking. Literature claims still belong in `02_literature/claims_ledger.csv`, and experimental results still belong in `04_experiments/experiment_registry.csv`.
