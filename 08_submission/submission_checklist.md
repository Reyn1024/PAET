# Submission Checklist

## Before Journal Selection

- Generic SCI manuscript complete.
- All manuscript citations verified.
- All claims registered.
- All results traceable to experiment records.
- Figures and tables registered.
- Limitations section complete.

## After Journal Selection

- Journal scope checked.
- Article type confirmed.
- Formatting guide downloaded.
- Reference style adapted.
- Word/figure/table limits checked.
- Data availability statement prepared.
- Ethics/privacy statement prepared.
- Conflict of interest statement prepared.
- Cover letter drafted.
- Highlights prepared if required.
- Graphical abstract prepared if required.
